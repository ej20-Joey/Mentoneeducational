How to Use
Step 1
Update css/tokens.css
All design token values live here. Inject Figma values via Claude Code — everything in the library updates automatically.
Step 2
Link the CSS files you need
Each component has its own CSS file. Only include what a page actually uses.
Step 3
Copy HTML patterns from component pages
Each component page shows all variants with their class names. Copy the HTML directly into your page templates.
