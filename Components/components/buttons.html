Mentone / Buttons
← Library Index
Variants

Core button variants. Use btn-primary for the main action per screen. btn-accent for high-emphasis commerce CTAs like "Add to Cart". Avoid more than one primary per view.

.btn .btn-primary / .btn-accent / .btn-outline / .btn-ghost / .btn-danger
Sizes

Four sizes. Default (no size modifier) is the standard for most UI. btn-lg for hero CTAs and checkout flows.

.btn-xs / .btn-sm / (default) / .btn-lg / .btn-xl
States

Disabled and loading states. Always apply disabled attribute for accessibility, not just the CSS class.

[disabled] / .btn-loading
Full Width

Use btn-full for mobile contexts, checkout flows, and form submissions.

.btn-full
On Dark Background
.btn-outline-white — use on dark/primary backgrounds
Icon Buttons

Square icon-only buttons for toolbar actions. Pair with a title attribute for accessibility.

.btn-icon / .btn-icon-round
